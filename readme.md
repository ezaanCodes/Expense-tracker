## ‚ö° Quick Start

### 1. Clone the repo

```sh
git clone https://github.com/yourusername/pernstackExpenseTracker.git
cd pernstackExpenseTracker
```

### 2. Install dependencies

```sh
cd backend
pnpm install
cd ../frontend
pnpm install
```

### 3. Setup environment

- Create `.env` in `backend/` with your DB and JWT secrets.

### 4. Run the app

**Backend:**
```sh
pnpm start
```

**Frontend:**
```sh
pnpm dev
```
